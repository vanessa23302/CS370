# CS 370 Project Two: Pirate Intelligent Agent

## Project Overview
In this project, I developed a pirate intelligent agent using deep Q-learning to solve a pathfinding problem. The goal of the agent was to navigate a maze and find the treasure by learning from rewards and penalties.

## What Code Was Provided vs Created
The starter code included the maze environment and the experience replay system. I was responsible for completing the Q-training algorithm. I implemented the deep Q-learning logic, including the training loop, epsilon-greedy action selection, experience replay, and target network updates. I also modified the training to use random starting positions so the agent could learn the entire maze instead of a single path.
